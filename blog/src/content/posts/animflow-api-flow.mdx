## 핵심 포인트

1. **비동기 처리**: `fetch()` API는 Promise를 반환하여 비동기적으로 작동합니다
2. **HTTP 메서드**: GET, POST, PUT, DELETE 등 다양한 메서드로 CRUD 작업을 수행
3. **상태 코드**: 200(성공), 404(찾을 수 없음), 500(서버 오류) 등으로 결과를 알림

## 코드 예시

```javascript
async function fetchUsers() {
  try {
    const response = await fetch('/api/users');
    if (!response.ok) {
      throw new Error(`HTTP error! status: ${response.status}`);
    }
    const users = await response.json();
    return users;
  } catch (error) {
    console.error('Failed to fetch users:', error);
  }
}
```

다음 포스트에서는 POST 요청과 에러 핸들링 패턴을 더 자세히 다루겠습니다.
