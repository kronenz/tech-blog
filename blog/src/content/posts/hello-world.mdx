# 안녕하세요!

이 블로그는 **AnimFlow**를 활용한 인터랙티브 기술 블로그입니다.

## 이 블로그의 특징

1. **인터랙티브 다이어그램**: AnimFlow DSL을 사용하여 애니메이션이 있는 기술 다이어그램을 제공합니다.
2. **MDX 기반**: Markdown과 React 컴포넌트를 함께 사용할 수 있습니다.
3. **정적 사이트**: Astro를 사용하여 빠른 로딩 속도를 제공합니다.

## 코드 예시

다음은 TypeScript 코드 예시입니다:

```typescript
interface DiagramConfig {
  title: string;
  nodes: Node[];
  edges: Edge[];
}

function createDiagram(config: DiagramConfig): Diagram {
  const diagram = new Diagram(config.title);

  for (const node of config.nodes) {
    diagram.addNode(node);
  }

  for (const edge of config.edges) {
    diagram.addEdge(edge);
  }

  return diagram;
}
```
