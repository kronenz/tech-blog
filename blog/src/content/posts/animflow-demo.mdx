# AnimFlow 다이어그램 데모

이 포스트에서는 AnimFlow를 사용하여 인터랙티브 다이어그램을 만드는 방법을 보여드립니다.

## 캐시 시스템 동작

아래 다이어그램은 캐시 시스템의 동작 원리를 보여줍니다:

<AnimFlowEmbed
  id="cache-demo"
  title="Cache Lookup Flow"
  height={450}
  yaml={`version: "1.0"
nodes:
  - id: client
    type: box
    label: Client
    position: { x: 100, y: 200 }
  - id: cache
    type: database
    label: Redis Cache
    position: { x: 350, y: 200 }
  - id: db
    type: database
    label: Database
    position: { x: 600, y: 200 }
edges:
  - id: e1
    from: client
    to: cache
    label: GET
  - id: e2
    from: cache
    to: db
    label: Query
  - id: e3
    from: db
    to: cache
    label: Result
  - id: e4
    from: cache
    to: client
    label: Response
scenarios:
  - id: cache-hit
    name: Cache Hit
    steps:
      - action: animate-edge

        label: "GET user:123"

      - action: highlight

          color: "#4ade80"

      - action: highlight

## 사용 방법

1. **Play** 버튼을 클릭하여 애니메이션을 시작합니다
2. 시나리오 드롭다운에서 "Cache Hit" 또는 "Cache Miss"를 선택합니다
3. **Speed** 드롭다운으로 재생 속도를 조절합니다
4. **Reset** 버튼으로 다이어그램을 초기 상태로 되돌립니다

## AnimFlow DSL 설명

AnimFlow는 YAML 기반의 선언적 DSL을 사용합니다:

- **nodes**: 다이어그램의 노드(엔티티)를 정의합니다
- **edges**: 노드 간의 연결을 정의합니다
- **scenarios**: 애니메이션 시나리오를 정의합니다
- **steps**: 각 시나리오의 애니메이션 단계를 정의합니다

자세한 내용은 [AnimFlow 문서](/docs/animflow)를 참조하세요.
